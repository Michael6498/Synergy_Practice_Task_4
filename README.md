# Synergy_Practice_Task_4
# Информационная система «TrueTravel»
### Кейс-задача № 4 по технологической практике

## Описание проекта
Данный проект представляет собой веб-ориентированную информационную систему для автоматизации работы туристического агентства. Приложение позволяет управлять базой данных стран и отелей, а также оформлять бронирования с автоматическим расчетом стоимости.

## Стек технологий
* **Backend:** Node.js (платформа), Express (веб-фреймворк).
* [cite_start]**Database:** MS SQL Server (хранение данных о турах, клиентах и заказах)[cite: 31].
* **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS (стилизация интерфейса).
* [cite_start]**Архитектура:** Трехзвенная WEB-архитектура (клиент — сервер приложений — сервер базы данных)[cite: 28].

## Функциональные возможности
* **Управление справочниками:** просмотр и редактирование данных о странах и отелях.
* **Калькулятор туров:** автоматический расчет итоговой цены в зависимости от параметров проживания.
* **Система бронирования:** создание и мониторинг заказов в режиме реального времени.
* **Безопасность:** централизованное хранение данных и обработка запросов на стороне сервера.

## Инструкция по развертыванию
1. **База данных:**
   * Откройте MS SQL Server Management Studio.
   * [cite_start]Выполните SQL-скрипт (находится в репозитории) для создания структуры таблиц «Туризм»[cite: 22, 23].
2. **Серверная часть:**
   * Склонируйте репозиторий: `git clone https://github.com/Michael6498/Synergy_Practice_Task_4.git`
   * Перейдите в папку проекта и установите зависимости: `npm install`
   * Настройте параметры подключения к вашей БД в файле `server.js`.
   * Запустите сервер: `node server.js`
3. **Клиентская часть:**
   * Откройте файл `index.html` в любом современном браузере.
